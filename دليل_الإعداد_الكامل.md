## خطوات التثبيت الكاملة

### ✅ الخطوة 1: تجهيز المجلدات

```powershell
# أنشئ المجلدات التالية (أو استخدم السكريبت المرفق)
mkdir C:\autosystem
mkdir C:\Users\HP\server_files
mkdir C:\Users\HP\server_files\audio
mkdir C:\Users\HP\server_files\videos
mkdir C:\Users\HP\server_files\subtitles
mkdir C:\Users\HP\server_files\exports
mkdir C:\Users\HP\server_files\bg_videos
mkdir C:\Users\HP\server_files\thumbnails
mkdir C:\Users\HP\server_files\music
mkdir C:\Users\HP\server_files\sfx
```

### ✅ الخطوة 2: وضع ملفات النظام

انسخ الملفات التالية إلى `C:\autosystem\`:
- `server.py` — خادم Flask الرئيسي
- `workflow_youtube_pro.json` — سير عمل n8n
- `tts_wrapper.py` — محسن TTS
- `check_video.py` — فحص الفيديو
- `create_video.py` — اختبار توليد فيديو
- `disable_sleep.bat` — منع السكون
- `SETUP_RUN_ONCE.bat` — إعداد مرة واحدة

### ✅ الخطوة 3: إعداد Pexels API

1. افتح حساب مجاني على: https://www.pexels.com/api/
2. احصل على API key
3. ضع المفتاح في `server.py`:

```python
PEXELS_API_KEY = "YOUR_PEXELS_API_KEY"  # غيّر هذا
```

### ✅ الخطوة 4: إعداد Google Sheets API

1. افتح: https://console.cloud.google.com/
2. أنشئ مشروع جديد → فعّل Google Sheets API
3. أنشئ Service Account → حمل JSON file
4. شارك Google Sheet مع البريد الإلكتروني للحساب
5. ضع ملف JSON في `C:\autosystem\` كـ `google-credentials.json`

### ✅ الخطوة 5: إعداد YouTube API

1. افتح: https://console.cloud.google.com/
2. فعّل YouTube Data API v3
3. أنشئ OAuth 2.0 credentials (Desktop app)
4. حمّل JSON → ضعه في `C:\autosystem\`
5. في n8n، أضف credential من نوع YouTube OAuth2

### ✅ الخطوة 6: إعداد n8n

```powershell
# شغّل n8n
n8n start

# افتح المتصفح: http://localhost:5678
# أنشئ حساب مشرف
# اذهب إلى Settings → Import → اختر workflow_youtube_pro.json
```

### ✅ الخطوة 7: إعداد Google Sheet للمحتوى

أنشئ Google Sheet جديد (أو استخدم الموجود) بالأعمدة التالية:

| topic | contentType | targetWords | keywords | tone | language |
|-------|-------------|-------------|----------|------|----------|
| الذكاء الاصطناعي | tutorial | 1000 | AI, ذكاء اصطناعي, تعلم | professional | ar |
| أفضل هواتف 2025 | review | 800 | هواتف, تقييم, مراجعة | conversational | ar |
| كيفية الربح من الإنترنت | tutorial | 1200 | ربح, إنترنت, عمل حر | motivational | ar |

ضع `YOUR_GOOGLE_SHEET_ID` في عقدة Google Sheets في n8n.

### ✅ الخطوة 8: إعداد WhatsApp (اختياري)

1. افتح: https://www.callmebot.com/blog/free-api-whatsapp-messages/
2. أرسل "I allow callmebot to send me messages" إلى الرقم +34 644 52 69 41
3. انتظر رسالة التأكيد
4. ضع المفاتيح في `server.py`:

## YouTube SEO والامتثال للقواعد

### ⚖️ الامتثال لسياسات يوتوب

النظام مصمم ليكون متوافقاً تماماً مع قواعد يوتوب:

| القاعدة | كيف يطبقها النظام |
|---------|-------------------|
| **محتوى أصلي** | كل سكريبت يُولد من الصفر (قوالب متغيرة) |
| **عدم التكرار** | Anti-fingerprinting: كل فيديو له بصمة فريدة |
| **عدم التضليل** | العناوين دقيقة وتعكس محتوى الفيديو فعلاً |
| **حقوق الطبع** | مشاهد Pexels مرخصة تجارياً |
| **الموسيقى** | موسيقى خالية من حقوق النشر |
| **Shorts** | مقاطع أصلية من الفيديو الطويل |
| **محتوى مناسب** | جميع القوالب تراعي الذوق العام |

### 🏆 تحسين أداء الخوارزمية

```
عنوان جذاب ← أول 60 حرفاً يحتوي الكلمة المفتاحية
وصف محسّن ← 200+ كلمة مع روابط وهاشتاقات
صورة مصغرة ← عالية التباين مع نص واضح
توقيت النشر ← أفضل وقت حسب نوع المحتوى
Chapters ← توقيتات في الوصف
تفاعل ← أسئلة في نهاية الفيديو
Shorts ← لزيادة الوصول
جدول ثابت ← 2 فيديو أسبوعياً على الأقل
```

### 🎯 معدل الاحتفاظ (Retention)

لتحسين retention:
1. **الـ Hook**: أول 15 ثانية تحتوي على وعد واضح
2. **الإيقاع**: تغيير المشهد كل 8-12 ثانية (Ken Burns)
3. **التفاعل**: أسئلة كل 3-4 دقائق
4. **Shorts**: مقاطع قصيرة من نفس المحتوى
5. **Chapters**: تسهيل التنقل بين الأجزاء

### 🔒 Anti-Fingerprinting (تشتيت البصمة)

النظام يغيّر كل مرة:
- FPS (23.976 / 24 / 25 / 29.97 / 30)
- دقة وهمية (±2-6 بيكسل)
- Gamma/Brightness/Contrast
- Colormatrix (bt709 / bt470bg / smpte170m / bt2020nc)
- Pixel format (yuv420p / yuv422p / yuv444p)
- CRF (20-24)
- Pitch/Speed audio طفيف
- ترتيب الفقرات
- أسلوب السكريبت (10 أنماط)
- صوت المقدم (8 أصوات مختلفة)

## تشغيل النظام تلقائياً

### 🚀 التشغيل الأولي

```powershell
# 1. شغّل السيرفر
cd C:\autosystem
python server.py

# 2. في نافذة أخرى، شغّل n8n
n8n start

# 3. افتح http://localhost:5678
# 4. Activate الـ workflow
```

### 🔄 التشغيل التلقائي مع Windows

```powershell
# شغّل السكريبت المرفق كـ Administrator
C:\autosystem\SETUP_RUN_ONCE.bat
```

هذا السكريبت:
- ينسخ الملفات إلى المجلدات الصحيحة
- يثبت المكتبات المطلوبة
- يضيف autostart.bat إلى Startup
- يشغّل disable_sleep.bat (يمنع السكون)
- يشغّل النظام فوراً

### 👁️ Watchdog (مراقب السيرفر)

`watchdog.bat` يراقب السيرفر كل 30 ثانية:
- إذا توقف → يعيد تشغيله تلقائياً
- يسجل في `watchdog.log`
- يعمل في الخلفية بدون تدخل

## استكشاف الأخطاء وإصلاحها

### ❌ مشكلة: السيرفر لا يعمل

```powershell
# تحقق من أن السيرفر شغال
curl http://127.0.0.1:5000/health

# افحص الـ logs
Get-Content C:\Users\HP\server_files\server.log -Tail 20

# تأكد من Python والمكتبات
python --version
pip list | findstr flask
```

### ❌ مشكلة: TTS يفشل

```powershell
# تحقق من الـ DNS (قد يكون محجوباً)
nslookup speech.platform.bing.com

# افحص tts_debug.log
Get-Content C:\autosystem\tts_debug.log -Tail 50

# جرّب يدوياً (لسرعة التشخيص)
python -c "import edge_tts; import asyncio; asyncio.run(edge_tts.Communicate('اختبار', 'ar-SA-ZariyahNeural').save('test.mp3'))"
```

إذا كان DNS محجوباً: السيرفر يستخدم Google DoH (DNS-over-HTTPS) للتجاوز.

### ❌ مشكلة: Pexels يفشل

```powershell
# تحقق من API key
curl -H "Authorization: YOUR_API_KEY" https://api.pexels.com/v1/popular?per_page=1

# تأكد من عدم استنزاف الحصة المجانية
# الحساب المجاني: 200 طلب/ساعة + 20000 طلب/شهر
```

### ❌ مشكلة: FFmpeg لا يعمل

```powershell
# تحقق من التثبيت
ffmpeg -version

# تأكد من وجوده في PATH
where ffmpeg
```

إذا لم يكن في PATH: غيّر المسار في `server.py`:

```python
FFMPEG_PATH = r"C:\ffmpeg\bin\ffmpeg.exe"
```

### ❌ مشكلة: YouTube API يرفض الرفع

تحقق من:
1. OAuth 2.0 credentials صالحة
2. الحصة اليومية لـ YouTube API (10000 وحدة/يوم)
3. الفيديو يتبع سياسات المحتوى
4. القناة مفعّلة وليس عليها قيود
