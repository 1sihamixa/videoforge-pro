🚀 دليل نشر VideoForge Pro على الإنترنت

اتبع هذه الخطوات لرفع الموقع على الإنترنت والربح من الإعلانات

📋 الخطوات الأساسية

1 تجهيز المفاتيح والبيانات

افتح ملف .env واملأ البيانات التالية:

# مفتاح API لخدمة sync.so (للتوليد السحابي)
SYNC_API_KEY=your_sync_api_key_here

# مفتاح Google AdSense (للربح من الإعلانات)
ADSENSE_CLIENT=ca-pub-xxxxxxxxxxxxxx
ADSENSE_SLOT_TOP=1234567890
ADSENSE_SLOT_SIDEBAR=1234567890
ADSENSE_SLOT_IN_ARTICLE=1234567890
ADSENSE_SLOT_VIDEO=1234567890

# Google Analytics (لتتبع الزوار)
GA_TRACKING_ID=G-XXXXXXXXXX

# مفتاح سري للتطبيق
SECRET_KEY=put-a-random-secret-key-here
2 رفع الملفات إلى منصة الاستضافة

اختر إحدى منصات الاستضافة المجانية أو المدفوعة:

🌐 Render (مجاني - موصى به)

1. ادخل على https://render.com
2. سجل حساب جديد (GitHub)
3. اضغط New > Web Service
4. اربط مستودع GitHub
5. اختر:
   - Name: videoforge-pro
   - Runtime: Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: gunicorn app:app --bind 0.0.0.0:$PORT --workers 2
6. أضف المتغيرات من .env
7. اضغط Create Web Service

☁️ Heroku

1. تأكد من وجود ملفات: Procfile, runtime.txt, requirements.txt
2. ارفع المشروع إلى GitHub
3. اربط مع Heroku واختر Python buildpack
4. أضف config vars من .env
5. انشر!

🖥️ خادم خاص (VPS)

# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل باستخدام gunicorn
gunicorn app:app --bind 0.0.0.0:8080 --workers 4 --timeout 300

# أو باستخدام PM2 مع عملية خلفية
npm install -g pm2
pm2 start "gunicorn app:app --bind 0.0.0.0:8080 --workers 4 --timeout 300" --name videoforge
3 إعداد Google AdSense

للربح من الإعلانات:

1. ادخل https://adsense.google.com
2. سجل حساب AdSense
3. أضف موقعك: https://your-domain.com
4. انتظر الموافقة (قد يستغرق أيام)
5. بعد الموافقة، أنشئ وحدات إعلانية:
   - إعلان علوي (Leaderboard) ← ADSENSE_SLOT_TOP
   - إعلان جانبي (Rectangle) ← ADSENSE_SLOT_SIDEBAR
   - إعلان داخل المحتوى ← ADSENSE_SLOT_IN_ARTICLE
   - إعلان سفلي ← ADSENSE_SLOT_VIDEO
6. انسخ client ID (ca-pub-...) وضعه في .env
4 إعداد النطاق (Domain) 
1. اشترِ نطاقاً مثل videoforge-pro.com
2. اربطه مع منصة الاستضافة (Render/Heroku/VPS)
3. فعّل HTTPS (شهادة SSL)
4. انتظر حتى ينتشر DNS (24-48 ساعة)
5 تحسين محركات البحث (SEO)

الموقع مضمن بالفعل علامات SEO الأساسية. أضف الموقع إلى:

1. Google Search Console - https://search.google.com/search-console
2. أضف sitemap.xml (سيفة موجودة تلقائياً)
3. أرسل الموقع للفهرسة
4. راقب الأداء في Google Analytics
